Vercel's Elastic Build Machines now automatically monitor memory usage and adjust machine tiers to prevent out-of-memory failures, upgrading when builds are memory-intensive or downgrading after successful deployments. This feature can be enabled in team or project settings and balances deployment reliability and cost by only tracking application memory, not Vercel's infrastructure usage.
