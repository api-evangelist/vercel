Alibaba's Qwen 3.7 Plus is now available on Vercel AI Gateway, offering unified vision and language capabilities for agent tasks, with both the Plus and Max versions free for paid Gateway users through June 4, 2026. The AI Gateway platform provides unified API access, usage tracking, and optimization features including dynamic provider sorting by latency and cost with no platform fees on inference.
