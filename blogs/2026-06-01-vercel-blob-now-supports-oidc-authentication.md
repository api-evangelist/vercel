Vercel Blob now defaults to OIDC authentication for new projects, replacing long-lived tokens with short-lived credentials that rotate automatically. Existing stores can upgrade through the Projects tab, enabling seamless authentication for both SDK functions and CLI operations without requiring permanent access tokens.
