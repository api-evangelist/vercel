The skills.sh API is now accessible for querying over 600,000 open-source skills using Vercel's short-lived OIDC tokens, which rotate automatically without requiring manual secret management. The service enforces a rate limit of 600 requests per minute per team and project, offering capabilities including skill searches, detailed information retrieval, and security audits.
